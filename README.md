# 版本比较

- **Stable**版本是标准版，更新慢，BUG少。

- **Beta**版本是测试版，更新偏快，但是也会有BUG和好玩的新功能。

- **Snapshot**版本是快照版，更新可能极快，BUG较多，但也会有“亦可赛艇”的新功能！
